Class: PriorityQueue

PriorityQueue

A queue backed by a Binary Heap. Basic queue operation offer and poll run in O(lgn) time. The elements in this queue are ordered according to their natural ordering or based on the compare function provided. Reference: PriorityQueue in Oracle JDK

new PriorityQueue(compareFn)

Example -

var PriorityQueue = require("dsjslib").PriorityQueue
var pq=new PriorityQueue(function(w1,w2){
     return w1.weight - w2.weight;
})
